- JavaScript

## 패키지 구성

```
frontend-master
├─ node_modules                        # 설치된 패키지
│   └─ ...
│
├─ public                              # 정적 파일 (예: 로고, favicon, Open Graph 이미지 등)
│   └─ ...
│
├─ src                                 # 주요 코드 폴더
│   ├─ assets                          # 애플리케이션 내부에서 동적으로 불러올 데이터(이미지 등)
│   │   ├─ image                       # 이미지 저장 폴더
│   │   └─ ...
│   │
│   ├─ components                      # 재사용 컴포넌트 폴더
│   │   ├─ footer                      # footer 컴포넌트
│   │   │   └─
│   │   ├─ header                      # header 컴포넌트
│   │   │   └─
│   │   └─ ...
│   │
│   ├─function                         # 공통으로 사용할 함수(.js 파일)
│   │   └─ ...
│   │
│   ├─ hooks                           # 공통으로 사용할 훅 (커스텀 훅)
│   │   └─ ...
│   │
│   ├─ page                            # 개별 페이지 컴포넌트 폴도
│   │   └─ ...
│   │
│   ├─ store                           # 상태 관리 폴더
│   │   └─ ...
│   │
│   ├─ services                        # API 등 서비스 로직 (사용 라이브러리에 따라 이름 변경 가능성 있음)
│   │   └─ ...
│   │
│   ├─ style                           # 스타일 파일 모음
│   │   └─ ...
│   │
│   ├─ App.jsx                         # 메인 컴포넌트
│   └─ Main.jsx                        # React 진입점
│
│
├─ .gitignore                          # git 제외 파일
├─ .env                                # 환경 변수 파일 (ex. API KEY 등) !!! 공유 금지 파일로 .gitignore 추가 !!!
├─ eslint.config.js                    # ESLint 설정 파일
├─ index.html                          # 프로젝트의 기본 HTML 파일
├─ package.json                        # 프로젝트 설정 및 종속성 목록
├─ eslint.config.js                    # ESLint 설정 파일
└─ vite.config.js                      # vite 설정 파일

```
